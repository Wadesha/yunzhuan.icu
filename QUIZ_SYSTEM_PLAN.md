## 5. 刷题交互设计

### 流程（重构后）

```
课程页点击"开始刷题"
    ↓
直接进入第1题（无开始页）
    ↓
阅读题干（可点"朗读"按钮）
    ↓
点击选项 / 键盘 1-4 选择
    ↓
立即显示对错反馈
    ↓
2秒后自动跳下一题
    ↓
第10题答完 → 结果页
    ↓
显示得分、解锁勋章
```

### 第1题界面

```
┌────────────────────────────────────┐
│  [← 返回代数基础]      1/10  XP:0  │  顶部
├────────────────────────────────────┤
│  题目                              │
│  栈（Stack）的特点是什么？         │  题干
│  [朗读]                            │  朗读按钮
│                                    │
│  ┌────────────────────────────┐    │
│  │ ① 先进先出（FIFO）          │    │  选项
│  └────────────────────────────┘    │  (键盘1)
│  ┌────────────────────────────┐    │
│  │ ② 先进后出（LIFO）          │    │  (键盘2)
│  └────────────────────────────┘    │
│  ┌────────────────────────────┐    │
│  │ ③ 随机访问                  │    │  (键盘3)
│  └────────────────────────────┘    │
│  ┌────────────────────────────┐    │
│  │ ④ 双向访问                  │    │  (键盘4)
│  └────────────────────────────┘    │
└────────────────────────────────────┘
```

**核心改动**：
- 移除"开始答题"按钮
- 顶部"返回课程"链接
- 朗读按钮位于题干下方
- 键盘 1-4 选择
- 答后2秒自动跳下一题

### 朗读功能
- 使用浏览器原生 `SpeechSynthesisUtterance` API
- 朗读内容：**仅题目题干**，不读选项
- 默认中文，英文题用英文
- 状态：`朗读` / `朗读中...` / `停止`

### 键盘快捷键

| 按键 | 功能 |
|------|------|
| `1` `2` `3` `4` | 选择选项 |
| `Enter` / `→` | 下一题 |
| `Space` | 朗读题干 |
| `Esc` | 退出刷题 |

### 结果页

```
┌────────────────────────────────────┐
│            80%                     │
│            8/10 正确               │
│                                    │
│  +80 XP                            │
│  +40 金币                          │
│                                    │
│  解锁成就: 首次及格                 │
│                                    │
│  [继续刷题]  [返回课程]             │
└────────────────────────────────────┘
```

## 6. 数值系统设计（核心）

### 6.1 核心数值

| 数值 | 用途 | 获取 |
|------|------|------|
| 经验值 XP | 升级、段位 | 答题 |
| 金币 Coin | 兑换/成就指标 | 答题 |
| 等级 Level | 段位显示 | XP累计 |
| 段位 Rank | 稀有度 | 等级映射 |
| 连击 Streak | 单次会话 | 连续答对 |
| 答题数 | 累计统计 | 任意答 |
| 正确数 | 累计统计 | 答对累计 |

### 6.2 经验值（XP）公式

```
单题XP = 基础值 × 难度系数 × 连击加成
```

| 难度 | 系数 | 答对 | 答错 |
|------|------|------|------|
| 简单 (1) | 1.0 | 10 | 2 |
| 中等 (2) | 1.5 | 15 | 3 |
| 困难 (3) | 2.0 | 20 | 4 |

**连击加成**：`1 + min(连击, 10) × 0.05`，最高1.5倍

### 6.3 金币公式

| 难度 | 答对 | 答错 |
|------|------|------|
| 简单 | 5 | 1 |
| 中等 | 8 | 1 |
| 困难 | 12 | 1 |

## 9. 奖励体系

### 9.1 闭环设计

```
答题 → XP/Coin → 升级/攒金币 → 解锁勋章/任务 → 成就感 → 继续刷题
```

### 9.2 金币消耗（v1）

v1 阶段金币**只进不出**，仅作成就感指标。v2 加入消耗渠道：
- 头像框
- 题目提示
- 跳过关卡
- 抽奖系统

### 9.3 等级特权（v1）

| 段位 | 解锁 |
|------|------|
| 白银I+ | 难题模式 |
| 黄金I+ | 限时挑战 |
| 铂金I+ | 排行榜（需后端） |
