回归分析

Regression Analysis

课程介绍 Course Introduction

学分：3 | 先修课：数理统计学、线性代数 | 学期：大二下学期

本课程系统讲授回归分析的理论与方法，是统计学专业最重要的应用课程之一。内容包括简单线性回归、多元线性回归、模型参数估计与假设检验、拟合优度检验、变量选择与模型诊断、多重共线性、异方差性、自相关、多项式回归、虚拟变量、交互效应、非线性回归、Logistic回归等。课程强调理论与实践结合，使用统计软件进行数据分析，培养学生建立和评估回归模型的能力。

This course covers the theory and practice of regression analysis, one of the most important applied statistics courses. Topics include simple and multiple linear regression, parameter estimation, hypothesis testing, goodness-of-fit, variable selection, model diagnostics, multicollinearity, heteroscedasticity, autocorrelation, polynomial regression, dummy variables, interaction effects, nonlinear regression, and logistic regression.

大作业 Final Project

作业标题：基于回归模型的数据分析与预测（Regression-Based Data Analysis and Prediction）

选取一个实际数据集，建立合适的回归模型（含变量选择与诊断），处理多重共线性或异方差等问题，进行预测与模型评估，撰写完整分析报告。

Choose a real dataset, build an appropriate regression model (with variable selection and diagnostics), handle multicollinearity or heteroscedasticity, perform prediction and model evaluation, and write a complete analysis report.
