统计计算

Statistical Computing

课程介绍 Course Introduction

学分：3 | 先修课：数理统计学、至少一门编程语言 | 学期：大三上学期

本课程讲授统计学中常用的计算方法和技术，是理论统计与实际应用之间的桥梁。内容包括随机数生成（逆变换法、接受拒绝法）、蒙特卡洛积分与模拟、自助法（Bootstrap）与刀切法（Jackknife）、EM算法、数值优化方法（梯度下降、牛顿法）、马尔可夫链蒙特卡洛（MCMC）、高维数据处理、并行计算基础等。课程主要使用R或Python语言，培养学生实现统计算法和处理实际数据的编程能力。

This course covers computational methods and techniques commonly used in statistics, bridging theoretical statistics and practical applications. Topics include random number generation, Monte Carlo integration and simulation, bootstrap and jackknife methods, EM algorithm, numerical optimization (gradient descent, Newton's method), MCMC, high-dimensional data processing, and parallel computing basics.

大作业 Final Project

作业标题：统计计算算法的实现与性能评估（Implementation and Performance Evaluation of Statistical Computing Algorithms）

选取一种统计计算算法（如Bootstrap、EM算法或MCMC），编程实现并进行数值实验，评估算法的精度、收敛性与计算效率，并应用于实际数据分析。

Choose a statistical computing algorithm (Bootstrap, EM, or MCMC), implement it, conduct numerical experiments to evaluate accuracy, convergence, and efficiency, and apply it to real data analysis.
