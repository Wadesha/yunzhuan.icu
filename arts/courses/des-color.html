色彩设计

Color Design

课程介绍 Course Introduction

学分：3 | 先修课：设计基础 | 学期：第二学期

本课程专注于设计领域的色彩应用理论与实践方法。课程内容涵盖色彩科学基础、色彩心理学、色彩文化学、色彩系统与色立体、配色原理与方法、色彩调和与对比等专业知识，特别注重色彩在品牌设计、界面设计、环境设计等不同设计领域的具体应用。通过色彩实验、配色练习和项目实践，学生将掌握科学的色彩分析方法和创造性的色彩运用能力，学会运用色彩传递信息、营造氛围、塑造品牌形象，为各设计方向的专业实践提供色彩策略支撑。

This course focuses on color application theory and practical methods in the design field. The curriculum covers professional knowledge including color science fundamentals, color psychology, color culture, color systems and color solids, color matching principles and methods, and color harmony and contrast, with special emphasis on the specific application of color in different design fields such as brand design, interface design, and environmental design. Through color experiments, color matching exercises, and project practice, students master scientific color analysis methods and creative color application abilities, learning to use color to convey information, create atmosphere, and shape brand image, providing color strategy support for professional practice in various design directions.

大作业 Final Project

作业标题：品牌色彩系统设计与应用

学生需为一个虚构的消费品牌设计完整的色彩系统，包含主色、辅助色、中性色、渐变色彩等完整色彩体系。要求对每个色彩进行CMYK、RGB、Hex、Pantone等多色值标注，并制定色彩使用规范。需将色彩系统应用于品牌标志、包装、网页界面、空间导视等至少五个不同的应用场景，展示色彩在不同媒介上的呈现效果。需提交色彩系统规范手册和应用效果图，阐述色彩选择的策略依据和情感表达意图。

Students design a complete color system for a fictional consumer brand, including a full color palette of primary colors, secondary colors, neutral colors, and gradient colors. Each color must be annotated with multiple color values including CMYK, RGB, Hex, and Pantone, and color usage specifications must be established. The color system must be applied to at least five different application scenarios such as brand logo, packaging, web interface, and spatial wayfinding, demonstrating the presentation effect of colors on different media. A color system style guide and application renderings are submitted, explaining the strategic basis for color selection and emotional expression intent.
