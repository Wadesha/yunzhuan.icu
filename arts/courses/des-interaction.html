交互设计

Interaction Design

课程介绍 Course Introduction

学分：3 | 先修课：平面设计 | 学期：第四学期

本课程聚焦数字产品的交互设计理论与实践方法，培养学生创建有效、易用且愉悦的用户体验的能力。课程内容涵盖用户研究方法、信息架构设计、交互设计原则、原型设计方法、可用性测试、设计规范等核心知识，同时介绍Figma、Axure等主流设计工具。通过完整的项目流程实践，学生将学习从用户需求分析到交互方案落地的全流程设计方法，掌握用户画像、用户旅程图、线框图、高保真原型等设计交付物的制作技能。课程强调以用户为中心的设计理念，注重培养学生的逻辑思维和问题解决能力。

This course focuses on interaction design theory and practical methods for digital products, cultivating students' ability to create effective, usable, and delightful user experiences. The curriculum covers core knowledge including user research methods, information architecture design, interaction design principles, prototyping methods, usability testing, and design systems, while introducing mainstream design tools such as Figma and Axure. Through complete project process practice, students learn the full design process from user needs analysis to interaction solution implementation, mastering skills for creating design deliverables such as user personas, user journey maps, wireframes, and high-fidelity prototypes. The course emphasizes user-centered design philosophy and focuses on developing students' logical thinking and problem-solving abilities.

大作业 Final Project

作业标题：交互原型设计

设计一个应用的交互流程，绘制交互原型，进行用户测试，优化交互体验。

Design interaction flows for an application, create interactive prototypes, conduct user testing, and optimize interaction experience.
