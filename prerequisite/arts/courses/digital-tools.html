数字工具

Digital Tools

课程介绍 Course Introduction

学分：2 | 建议学期：第1学期

数字工具学习设计和创作常用的软件。掌握图像处理、矢量绘图、排版设计等工具的基本操作，培养数字化创作能力。

Digital tools teaches common design and creation software. Learn image processing, vector graphics, and layout design to develop digital creation skills.

学习示例 Learning Example

📋 示例：制作一张社交媒体图片：用Photoshop调整图片大小和色彩，添加文字和图形元素，确保尺寸适合不同平台如Instagram、微博等。
📋 Example: Create a social media image: Use Photoshop to resize and adjust colors, add text and graphic elements, ensure dimensions fit platforms like Instagram and Weibo.
