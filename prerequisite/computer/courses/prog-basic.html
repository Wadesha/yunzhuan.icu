编程入门

Programming Basics

课程介绍 Course Introduction

学分：3 | 建议学期：第1学期

编程入门学习计算机编程的基本概念和技能。学习变量、数据类型、控制结构、函数、数组等基础语法，掌握用代码解决简单问题的能力。

Programming basics introduces fundamental concepts and skills. Learn variables, data types, control structures, functions, arrays, and problem-solving with code.

学习示例 Learning Example

📋 示例：写一个简单的猜数字游戏：程序随机生成一个1-100的数字，用户输入猜测，程序提示"太大"或"太小"，直到猜对为止。这个练习涵盖了随机数、输入输出和循环控制。
📋 Example: Build a number guessing game: Program generates random number 1-100, user guesses, program hints "too big" or "too small" until correct. This covers random numbers, I/O, and loop control.
