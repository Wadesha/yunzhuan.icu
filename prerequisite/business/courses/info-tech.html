信息技术基础

IT Fundamentals

课程介绍 Course Introduction

学分：2 | 建议学期：第1学期

信息技术基础介绍计算机硬件、软件、网络和数据库的基本概念。学习操作系统使用、办公软件、网络基础知识等实用技能。

IT fundamentals introduces computer hardware, software, networks, and databases. Learn OS usage, office software, and basic networking skills.

学习示例 Learning Example

📋 示例：配置一个简单的局域网：用路由器连接多台电脑，设置IP地址和子网掩码，实现文件共享和打印机共享。这是小型办公室的常见网络配置。
📋 Example: Configure a simple LAN: Connect computers with a router, set IP addresses and subnet masks, enable file and printer sharing. This is common in small office networks.
