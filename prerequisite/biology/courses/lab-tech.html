实验室技术

Lab Techniques

课程介绍 Course Introduction

学分：2 | 建议学期：第1学期

实验室技术学习科学实验的基本操作和安全规范。掌握实验仪器使用、数据记录、误差分析、报告撰写等技能。

Lab techniques teaches basic experimental operations and safety protocols. Learn instrument use, data recording, error analysis, and report writing.

提示：建议先理解本课程介绍和示例，然后点击下方按钮进行刷题自测。

学习示例 Learning Example

示例：测量液体密度：用天平称量液体质量，用量筒测量体积，然后计算密度。多次测量取平均值，评估误差来源如温度变化对密度的影响。
Example: Measure liquid density: Weigh liquid with balance, measure volume with graduated cylinder, calculate density. Average multiple measurements and evaluate error sources like temperature effects.
