软件测试与质量保证

Software Testing and Quality Assurance

课程介绍 Course Introduction

学分：3 | 先修课：软件工程导论、数据结构 | 学期：第5学期

软件测试与质量保证课程系统讲授软件测试的理论、方法与工具，旨在保障软件质量、降低缺陷率。主要内容包括黑白盒测试、单元测试、集成测试、系统测试、回归测试、性能测试、自动化测试框架（如JUnit、Selenium），以及代码覆盖率分析与持续集成。学生将掌握测试用例设计、缺陷管理流程与质量度量方法。

Software Testing and Quality Assurance systematically covers testing theory, methods, and tools to ensure software quality and reduce defects. Topics include black-box and white-box testing, unit, integration, system, regression, and performance testing, automation frameworks (JUnit, Selenium), coverage analysis, and continuous integration. Students master test case design, defect management, and quality metrics.

大作业 Final Project

作业标题：Web应用测试体系设计与自动化测试实现

为一个Web应用设计完整的测试体系，涵盖单元、集成、系统与性能测试，并使用JUnit/Selenium等框架实现自动化测试。提交测试方案、自动化脚本与质量报告。

Design a complete testing framework for a Web application, covering unit, integration, system, and performance testing, and implement automation using JUnit/Selenium. Deliver a test plan, automation scripts, and a quality report.

实施步骤 Implementation Steps

步骤 1
测试策略制定
分析被测系统，制定测试计划与覆盖率目标
产出：测试计划文档
步骤 2
测试用例设计
运用等价类、边界值等方法设计用例
产出：测试用例集
步骤 3
自动化实现
JUnit实现单元测试，Selenium实现UI自动化
产出：自动化脚本
步骤 4
性能测试
使用JMeter进行负载与压力测试
产出：性能测试报告
步骤 5
缺陷分析与文档
分析缺陷与覆盖率，撰写质量报告
产出：最终提交材料

Steps
