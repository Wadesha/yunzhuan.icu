系统分析与设计

System Analysis and Design

课程介绍 Course Introduction

学分：3 | 先修课：数据库管理 | 学期：第4学期

系统分析与设计课程讲授信息系统开发生命周期（SDLC）各阶段的方法与技术。主要内容包括需求获取与分析（访谈、问卷、用例）、结构化与面向对象分析方法、数据流图（DFD）、UML建模（用例图、类图、时序图）、系统设计（架构、数据库、界面）、敏捷方法与原型开发，以及实施与测试策略。培养学生从业务需求到系统方案的转化能力。

System Analysis and Design covers methods and techniques across the system development lifecycle (SDLC). Topics include requirements elicitation and analysis (interviews, surveys, use cases), structured and object-oriented analysis, data flow diagrams, UML modeling (use case, class, sequence diagrams), system design (architecture, database, interface), agile and prototyping, and implementation and testing strategies, enabling students to translate business needs into system solutions.
