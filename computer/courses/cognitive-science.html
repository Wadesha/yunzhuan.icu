认知科学

Cognitive Science

课程介绍 Course Introduction

学分：3 | 先修课：心理学基础 | 学期：第2学期

认知科学是研究心智与智能的跨学科课程，融合心理学、计算机科学、神经科学、语言学与哲学的视角。课程重点讲解认知架构、信息加工模型、表征与计算、问题求解与决策等核心议题，并探讨认知科学在人机交互、人工智能与学习科学中的应用。学生将通过案例与实验理解人类认知规律，指导智能化界面与系统的设计。

Cognitive Science is an interdisciplinary course studying mind and intelligence, integrating psychology, computer science, neuroscience, linguistics, and philosophy. Topics include cognitive architecture, information processing models, representation and computation, problem solving, and decision making, with applications in HCI, artificial intelligence, and learning science. Students understand human cognition to guide the design of intelligent interfaces.

大作业 Final Project

作业标题：认知建模与用户任务分析

学生选择一个真实交互任务（如在线购物流程或信息检索任务），运用GOMS或KLM认知建模方法对用户操作进行建模，预测任务完成时间与认知负荷，并结合实证数据验证模型，最终撰写分析报告，提出界面优化建议。

Students select a real interaction task such as online shopping or information retrieval, apply GOMS or KLM cognitive modeling methods to model user operations, predict task completion time and cognitive load, validate the model with empirical data, and produce an analysis report with interface optimization suggestions.
