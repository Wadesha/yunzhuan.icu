用户研究方法

User Research Methods

课程介绍 Course Introduction

学分：3 | 先修课：心理学基础 | 学期：第3学期

用户研究方法系统讲授如何科学地理解用户需求、行为与体验。课程涵盖定性研究（深度访谈、田野观察、焦点小组）与定量研究（问卷调查、A/B测试、日志分析）两大类方法，重点训练研究方案设计、抽样、数据收集与分析的完整流程。学生将学习用户画像构建、用户旅程地图等工具，将研究结果转化为设计洞察，支撑产品决策。

User Research Methods systematically teaches how to scientifically understand user needs, behaviors, and experiences. The course covers qualitative methods such as in-depth interviews, field observation, and focus groups, as well as quantitative methods including surveys, A/B testing, and log analysis. Students learn research design, sampling, data collection, and analysis, transforming findings into design insights.

大作业 Final Project

作业标题：用户访谈与研究报告

学生需围绕一款目标产品，招募至少8位真实用户开展半结构化访谈，记录并转录访谈内容，运用亲和图法（KJ法）进行主题编码分析，输出用户画像、痛点清单与设计机会点，最终提交完整用户研究报告并进行课堂汇报。

Students select a target product, recruit at least 8 real users for semi-structured interviews, transcribe and analyze the data using affinity diagramming (KJ method), produce user personas, pain point lists, and design opportunities, and submit a complete user research report with a class presentation.
