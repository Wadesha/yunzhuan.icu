软件项目管理

Software Project Management

课程介绍 Course Introduction

学分：3 | 先修课：软件工程导论 | 学期：第6学期

软件项目管理课程讲授软件项目从立项到交付全过程的计划、组织、监控与风险管理方法。主要内容包括项目范围与工作分解结构（WBS）、进度计划（甘特图、PERT/CPM）、成本估算（COCOMO模型）、敏捷与Scrum管理、团队协作与沟通、风险识别与应对、配置管理与版本控制。培养学生的项目管理思维与领导力。

Software Project Management covers planning, organization, monitoring, and risk management across the software project lifecycle. Topics include scope and WBS, scheduling (Gantt, PERT/CPM), cost estimation (COCOMO), agile and Scrum, team collaboration, risk identification and response, configuration management, and version control. Students develop project management thinking and leadership.

大作业 Final Project

作业标题：项目计划与模拟执行

学生以小组形式选择一个软件项目，制定完整的项目计划书，包含范围说明、WBS、甘特图、风险登记册与成本估算。在6周的模拟执行期内使用Scrum或看板方法跟踪进度，撰写周报与变更记录。最终提交项目计划书、执行报告与小组复盘总结，并进行答辩。

In teams, students select a software project and produce a complete project plan including scope statement, WBS, Gantt chart, risk register, and cost estimate. During a six-week simulated execution using Scrum or Kanban, teams track progress and write weekly reports and change logs. Submit the project plan, execution report, retrospective summary, and conduct a defense.
