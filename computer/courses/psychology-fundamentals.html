心理学基础

Psychology Fundamentals

课程介绍 Course Introduction

学分：3 | 先修课：无 | 学期：第1学期

心理学基础是人机交互专业的入门课程，系统介绍心理学的基本理论、研究方法与核心概念。课程涵盖感知与知觉、注意力与记忆、学习与思维、情绪与动机等内容，重点探讨人类信息加工过程及其对界面设计的启示。通过本课程，学生将理解用户行为背后的心理机制，为后续的用户研究、交互设计等课程奠定理论基础。

Psychology Fundamentals is an introductory course for HCI majors, systematically covering psychological theories, research methods, and core concepts. Topics include perception, attention, memory, learning, thinking, emotion, and motivation, with emphasis on human information processing and its implications for interface design. Students will understand the psychological mechanisms underlying user behavior.

大作业 Final Project

作业标题：用户行为心理机制分析与界面设计启示研究

选择一个人机交互场景，运用感知、注意、记忆等心理学理论分析用户行为机制，并提出界面设计改进建议。提交理论分析、用户实验与设计建议报告。

Select an HCI scenario, apply perception, attention, and memory theories to analyze user behavior mechanisms, and propose interface design improvements. Deliver a theoretical analysis, user experiment, and design recommendations report.
