交互设计

Interaction Design

课程介绍 Course Introduction

学分：3 | 先修课：用户研究方法 | 学期：第4学期

交互设计聚焦于人与产品之间交互行为的设计，强调"如何设计行为"。课程围绕需求分析、概念设计、原型构建与迭代评估展开，涵盖交互模式、信息架构、状态流转、反馈机制等核心知识。学生将学习从用户场景出发，构建合理、流畅、可学习的交互流程，掌握低保真到高保真原型工具，并在多轮迭代中打磨交互体验。

Interaction Design focuses on designing the interactive behaviors between humans and products. The course covers requirement analysis, conceptual design, prototyping, and iterative evaluation, including interaction patterns, information architecture, state transitions, and feedback mechanisms. Students learn to build smooth, learnable interaction flows from user scenarios, mastering low- to high-fidelity prototyping tools.

大作业 Final Project

作业标题：交互原型设计

设计一个应用的交互流程，绘制交互原型，进行用户测试，优化交互体验。

Design interaction flows for an application, create interactive prototypes, conduct user testing, and optimize interaction experience.
