数字电路设计

Digital Circuit Design

课程介绍 Course Introduction

学分：3 | 先修课：电路分析 | 学期：第4学期

数字电路设计是电气与电子类专业的核心基础课程，讲授数字逻辑与数字系统的分析设计方法。课程内容包括数制与码制、逻辑代数与化简、组合逻辑电路分析与设计、触发器与时序逻辑电路、计数器与寄存器、同步与异步状态机、可编程逻辑器件与硬件描述语言 Verilog HDL。通过本课程学习，学生能够独立完成中等复杂度数字系统的设计，为后续嵌入式与计算机体系结构课程奠定基础。

Digital Circuit Design is a core foundational course for electrical and electronic majors, teaching analysis and design methods for digital logic and systems. Topics include number systems and codes, Boolean algebra and simplification, combinational logic design, flip-flops and sequential circuits, counters and registers, synchronous and asynchronous state machines, programmable logic devices, and Verilog HDL. Students learn to design moderately complex digital systems, preparing for embedded systems and computer architecture.

大作业 Final Project

作业标题：多功能数字钟Verilog HDL设计与FPGA实现 / Multi-Function Digital Clock Verilog HDL Design and FPGA Implementation

使用Verilog HDL设计多功能数字钟，包括计时、校时与闹钟功能，完成状态机设计与FPGA综合下载验证。

Design a multi-function digital clock using Verilog HDL, including timing, setting and alarm functions, completing state machine design and FPGA synthesis download verification.

实施步骤 Implementation Steps

步骤 1
功能需求分析
确定计时、校时、闹钟等功能与输入输出接口
产出：功能规格说明书
步骤 2
系统架构设计
设计分频、计数、译码等模块划分
产出：系统框图
步骤 3
Verilog代码编写
编写各模块RTL代码与状态机
产出：Verilog源代码
步骤 4
仿真与综合
进行功能仿真与FPGA综合实现
产出：仿真波形与综合报告
步骤 5
下载验证与报告
FPGA开发板下载验证并撰写设计报告
产出：数字系统设计报告

Steps

Step 1
Functional Requirements Analysis
Define timing, setting, alarm functions and I/O interface
Deliverable: Functional Spec Document
Step 2
System Architecture Design
Design divider, counter and decoder module partition
Deliverable: System Block Diagram
Step 3
Verilog Code Writing
Write RTL code and state machines for modules
Deliverable: Verilog Source Code
Step 4
Simulation and Synthesis
Perform functional simulation and FPGA synthesis
Deliverable: Simulation Waveforms and Synthesis Report
Step 5
Download Verification and Report
FPGA board verification and write design report
Deliverable: Digital System Design Report
