行为矫正原理

Principles of Behavior Modification

课程介绍 Course Introduction

学分：3 | 先修课：特殊教育导论、普通心理学 | 学期：秋季

本课程系统介绍应用行为分析（ABA）的基本原理与行为矫正技术。课程内容包括行为的实验分析基础、经典条件作用与操作条件作用理论、强化与惩罚原理、行为的功能评估方法、以及各类行为改变技术（正强化、负强化、消退、塑造、渐隐、代币制、反应代价等）。学生将学习如何运用ABC行为分析模式识别问题行为的功能，设计和实施行为干预方案，并对干预效果进行评估。课程强调伦理原则和积极行为支持的理念。

This course systematically introduces the principles of Applied Behavior Analysis (ABA) and behavior modification techniques. Topics include the experimental analysis of behavior, classical and operant conditioning, reinforcement and punishment principles, functional behavioral assessment, and behavior change techniques (positive reinforcement, negative reinforcement, extinction, shaping, fading, token economy, response cost, etc.). Students learn to use the ABC model for functional analysis, design and implement behavior intervention plans, and evaluate intervention effectiveness, with emphasis on ethical principles and positive behavior support.

大作业 Final Project

作业标题：问题行为功能评估与干预方案

针对一名儿童的问题行为案例，运用ABC模式进行功能评估，设计基于积极行为支持的干预方案，并设定效果评估指标。

For a child's problem behavior case, conduct functional assessment using the ABC model, design an intervention plan based on positive behavior support, and set outcome evaluation indicators.
