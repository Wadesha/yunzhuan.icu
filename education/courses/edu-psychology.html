教育心理学

Educational Psychology

课程介绍 Course Introduction

学分：3 | 先修课：教育学导论 | 学期：春季

本课程运用心理学理论与方法研究教育过程中的心理现象及其规律。主要内容包括认知发展理论、学习理论（行为主义、认知主义、建构主义）、学习动机、智力与创造力、人格发展、课堂管理心理、学习评估等。课程注重理论联系实际，通过案例分析和小组讨论，培养学生运用教育心理学原理分析和解决教育实际问题的能力。

This course applies psychological theories and methods to study mental phenomena in educational contexts. Topics include cognitive development, learning theories (behaviorism, cognitivism, constructivism), motivation, intelligence and creativity, personality development, classroom management psychology, and learning assessment. Emphasis is placed on connecting theory with practice through case analysis and group discussion.

大作业 Final Project

作业标题：教育心理学案例分析报告

选择一个真实学习问题案例（如学习动机不足、课堂管理困难等），运用学习理论、动机理论等分析成因并提出干预方案，撰写分析报告。

Select a real learning problem case (e.g., low motivation, classroom management difficulties), analyze causes using learning and motivation theories, propose an intervention plan, and write an analysis report.
