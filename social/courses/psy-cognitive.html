认知心理学

Cognitive Psychology

课程介绍 Course Introduction

学分：3 | 先修课：心理学导论 | 学期：大三上

本课程系统介绍人类认知过程的心理学研究，探讨心智如何获取、存储、提取和运用知识。内容涵盖注意、知觉模式识别、记忆的结构与过程、知识表征与概念形成、语言理解与产生、问题解决与推理、决策与判断等。课程结合认知神经科学研究成果，介绍信息加工理论、联结主义模型等主要理论取向。

This course provides a systematic introduction to the study of human cognitive processes, exploring how the mind acquires, stores, retrieves, and uses knowledge. Topics include attention, perceptual pattern recognition, memory structures and processes, knowledge representation and concept formation, language comprehension and production, problem solving and reasoning, and decision making and judgment. Cognitive neuroscience findings and major theoretical frameworks such as information processing and connectionist models are integrated throughout.

大作业 Final Project

作业标题：经典认知实验验证与扩展研究

学生选择一个经典认知心理学实验（如Stroop效应、系列位置效应、心理旋转实验等），使用E-prime或Python编写实验程序，招募被试重复实验并尝试扩展一个变量。报告需包含文献综述、实验设计、程序描述、结果分析与讨论，字数约3000字，并附实验程序代码。

Students select a classic cognitive psychology experiment (e.g., Stroop effect, serial position effect, mental rotation experiment), program it using E-prime or Python, recruit participants to replicate the experiment and extend one variable. The report includes literature review, experimental design, program description, results analysis and discussion, approximately 3000 words, with experimental program code attached.
